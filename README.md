# MiniCore - Generador de Reportes de Tareas

Este proyecto es una aplicación web desarrollada con **Spring Boot** diseñada para la gestión y visualización de reportes de tareas técnicas.

## Funcionalidades Principales

*   **Filtrado de Tareas:** Permite al usuario consultar tareas en estado "En progreso" dentro de un rango de fechas específico.
*   **Cálculo de Retrasos:** El sistema calcula automáticamente los días de retraso basándose en la fecha estimada de finalización.
*   **Interfaz Responsiva:** Diseño limpio y sencillo utilizando CSS Vanilla para una mejor experiencia de usuario.
*   **Validación de Datos:** Incluye validaciones del lado del cliente mediante JavaScript para asegurar que los rangos de fechas sean lógicos.

## Tecnologías Utilizadas

*   **Backend:** Java 21, Spring Boot 4.0.5 (Web, Data JPA, Thymeleaf).
*   **Base de Datos:** Microsoft SQL Server.
*   **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
*   **Herramientas:** Maven, Lombok.

## Despliegue

La aplicación ha sido desplegada exitosamente en **Microsoft Azure**, utilizando **Azure App Service** para el alojamiento del servidor y **Azure SQL Database** para la persistencia de datos.
